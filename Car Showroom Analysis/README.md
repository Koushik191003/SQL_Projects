# Car Showroom Analysis
## Description
This project is a comprehensive SQL-based analysis of a car showroom dataset, which includes three main tables: Cars, Customers, and Sales. The aim of the project is to analyze sales performance, customer behavior, and inventory management to provide actionable business insights.

### ANALYSIS PROCESS
### Total Revenue Generated by Each Car Brand

Query :
````sql
SELECT 
	C.Brand ,
    Floor(SUM(S.Sale_Price * S.Quantity)) AS Total_Revenue
FROM Sales As S
LEFT JOIN CARS AS C ON S.car_ID = C.car_ID
GROUP BY C.Brand 
ORDER BY Total_Revenue DESC;
````

### Age Group-wise Sales Contribution

Query :
````sql
SELECT 
    CASE
        WHEN C.Age BETWEEN 18 AND 25 THEN '18-25'
        WHEN C.Age BETWEEN 26 AND 35 THEN '26-35'
        WHEN C.Age BETWEEN 36 AND 50 THEN '36-50'
        ELSE '50+'
    END AS Age_Group,
    COUNT(C.Customer_ID) AS Customers,
    FLOOR(SUM(S.Sale_Price * S.Quantity)) AS Total_Spent
FROM Customers AS C
JOIN Sales AS S ON C.Customer_ID = S.Customer_ID
GROUP BY Age_Group
ORDER BY Total_Spent DESC;
````
